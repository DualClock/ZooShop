# 🐾 ZooShop ИС

## Система управления зоомагазином

**Версия:** 1.0
**Дата:** Апрель 2026
**Платформа:** Windows (.NET 10.0, WPF)

---

## 📖 ОПИСАНИЕ

**ZooShop ИС** — это информационная система для автоматизации работы зоомагазина. Приложение позволяет управлять продажами, товарами, клиентами, питомцами, сотрудниками и складскими операциями.

### Назначение системы

- Автоматизация бизнес-процессов зоомагазина
- Оформление продаж и управление корзиной
- Учёт товаров, брендов, категорий и видов животных
- Ведение базы клиентов и их питомцев
- Управление складом и движениями товаров
- Формирование отчётов и аналитики

## ✨ ВОЗМОЖНОСТИ СИСТЕМЫ

### 🛒 Продажи

- Оформление продаж в один клик
- Добавление товаров из встроенного списка с поиском и фильтрами
- Создание нового клиента прямо в окне продажи
- Выбор сотрудника, склада и способа оплаты
- Автоматическая генерация чека
- Автосписание остатков со склада

### 📦 Каталог товаров

- Полный CRUD товаров (создание, редактирование, удаление)
- Создание новых **брендов**, **категорий** и **видов животных** прямо из формы товара
- Указание цены с сохранением истории
- Остатки по складам
- Поиск и фильтрация по бренду, категории, виду животного

###  Склад

- Просмотр остатков по каждому складу
- Оформление движений: приход, расход, списание, перемещение
- Привязка к документам и сотрудникам
- История последних 50 движений

### 👥 Сотрудники

- CRUD сотрудников с назначением роли (Админ / Сотрудник)
- Указание должности, телефона, email
- Деактивация вместо удаления

### 👨‍👩‍ Клиенты и питомцы

- Управление клиентской базой
- Привязка питомцев к клиентам (вид, порода, аллергии, пол)
- Быстрый поиск клиентов

### 📋 Задачи и расписание

- Создание задач с приоритетом и ответственным
- Отслеживание статуса (Новая, В работе, Выполнена)
- Расписание смен сотрудников

### 📊 Отчёты

- Отчёт по продажам за выбранный период
- Топ товаров по выручке
- Статистика по сотрудникам
- Экспорт в CSV

### 📞 Поддержка

- Форма обратной связи
- Раздел FAQ

## 🚀 БЫСТРЫЙ СТАРТ

### Установка за 3 шага через PowerShell

> Откройте **PowerShell** и выполните следующие команды:

#### 1. Перейдите в удобную директорию

```powershell
cd C:\Users\YourName\Desktop
```

> Замените `YourName` на ваше имя пользователя.

#### 2. Клонируйте проект

```powershell
git clone https://github.com/DualClock/ZooShop.git ZooShop
cd ZooShop\ZooShop
```

#### 3. Примените миграции базы данных

```powershell
dotnet ef database update
```

> Убедитесь что на компьютере установлен **SQL Server (LocalDB)**. Он входит в состав Visual Studio и устанавливается автоматически с .NET SDK.

#### 4. Запустите приложение

```powershell
dotnet run
```

Или найдите `ZooShop.exe` в папке `bin\Debug\net10.0-windows\`.

## 📊 БАЗА ДАННЫХ

### Схема базы данных

```
ZooShopDB
├── AnimalTypes        # Виды животных
├── Categories         # Категории товаров
├── Brands             # Бренды
├── Warehouses         # Склады/магазины
├── Employees          # Сотрудники
├── Products           # Товары
├── PriceHistory       # История цен
├── Clients            # Клиенты
├── Pets               # Питомцы клиентов
├── InventoryMovements # Движения товаров
├── Sales              # Продажи (чеки)
├── SaleItems          # Строки чека
├── Tasks              # Задачи
├── Schedules          # Расписание смен
├── vw_CurrentStock    # Представление: текущие остатки
└── vw_TodaySales      # Представление: статистика за сегодня
```

### Основные таблицы

| Таблица | Описание | Ключевые поля |
|---------|----------|---------------|
| **Employees** | Сотрудники | EmployeeID, FirstName, LastName, Role, Position |
| **Clients** | Клиенты | ClientID, FirstName, LastName, Phone, Email |
| **Pets** | Питомцы | PetID, ClientID, PetName, AnimalTypeID, Breed |
| **Products** | Товары | ProductID, SKU, ProductName, BrandID, CategoryID |
| **PriceHistory** | История цен | PriceID, ProductID, Price, EffectiveDate |
| **Sales** | Продажи | SaleID, ReceiptNumber, ClientID, EmployeeID, TotalAmount |
| **SaleItems** | Строки чека | SaleItemID, SaleID, ProductID, Quantity, UnitPrice |
| **InventoryMovements** | Движения | MovementID, ProductID, WarehouseID, MovementType, Quantity |

## 🔧 КОНФИГУРАЦИЯ

Строка подключения задаётся в `ZooShopDbContext.cs`:

```csharp
public const string ConnectionString = @"Data Source=(localdb)\MSSQLLocalDB;Initial Catalog=ZooShop;Integrated Security=True;...";
```

### Параметры подключения

| Параметр | Значение по умолчанию | Описание |
|----------|----------------------|----------|
| Server | (localdb)\MSSQLLocalDB | Имя сервера SQL Server |
| Database | ZooShop | Имя базы данных |
| Integrated Security | True | Windows-аутентификация |
